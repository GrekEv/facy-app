  1. Получите токен бота от @BotFather
  2. Создайте .env файл (скопируйте .env.example)
  3. Добавьте BOT_TOKEN в .env
  4. Запустите: ./start.sh
  5. Откройте бота в Telegram
  6. Нажмите /start
  7. Откройте Web App

  Для продакшена:
  • Получите API ключи (deepfacevideo.com, ffans.ai)
  • Настройте домен и SSL
  • Используйте PostgreSQL вместо SQLite
  • Подключите Telegram Payments
