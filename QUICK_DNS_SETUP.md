# Настройка DNS для onlyface.art

## Где найти домены в Timeweb

1. https://timeweb.cloud/
2. "Домены и SSL" (левое меню)

## Подключение домена

1. "Домены и SSL" → "Добавить домен"
2. Введите: onlyface.art
3. Выберите: "Домен зарегистрирован у другого регистратора"
4. Скопируйте DNS-серверы

## Настройка DNS-серверов в reg.ru

1. reg.ru → "Домены" → onlyface.art → "DNS-серверы и управление зоной"
2. Измените DNS-серверы на серверы Timeweb
3. Сохраните

## Добавление A-записей в Timeweb

1. "Домены и SSL" → onlyface.art
2. Вкладка "DNS-записи"
3. "Добавить запись"

Запись 1:
- Тип: A
- Имя: @
- Значение: 72.56.85.215

Запись 2:
- Тип: A
- Имя: www
- Значение: 72.56.85.215

## Проверка

```bash
dig +short onlyface.art @8.8.8.8
dig +short www.onlyface.art @8.8.8.8
```

## После настройки DNS

```bash
ssh root@72.56.85.215
cd ~/facy-app
./setup_domain_onlyface.sh
```
