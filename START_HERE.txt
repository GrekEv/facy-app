🚀 БЫСТРЫЙ СТАРТ (3 ШАГА):
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  ШАГ 1: Получите токен бота
  ────────────────────────────────────────────────────────────────────────
    1. Откройте @BotFather в Telegram
    2. Отправьте /newbot
    3. Введите название бота
    4. Введите username бота (должен заканчиваться на bot)
    5. Скопируйте токен


  ШАГ 2: Настройте проект
  ────────────────────────────────────────────────────────────────────────
    В терминале:
    
    cd /Users/kirilldeniushkin/telegram-deepface-app
    cp .env.example .env
    
    Откройте файл .env и добавьте:
    BOT_TOKEN=ваш_токен_от_BotFather


  ШАГ 3: Запустите приложение
  ────────────────────────────────────────────────────────────────────────
    
    ВАРИАНТ А - Автоматический запуск:
    
      macOS/Linux:
      $ ./start.sh
      
      Windows:
      > start.bat
    
    ─────────────────────────────────────────────────────────────────────
    
    ВАРИАНТ Б - Ручной запуск (2 терминала):
    
      Терминал 1 - API сервер:
      $ python run_api.py
      
      Терминал 2 - Telegram бот:
      $ python main.py
    
    ─────────────────────────────────────────────────────────────────────
    
    ВАРИАНТ В - Docker:
    
      $ docker-compose up -d


  ГОТОВО! 🎉
  ────────────────────────────────────────────────────────────────────────
    1. Найдите вашего бота в Telegram
    2. Отправьте /start
    3. Нажмите "🚀 Открыть приложение"

🆘 ПРОБЛЕМЫ?
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  Бот не отвечает:
    ✓ Проверьте правильность BOT_TOKEN в .env
    ✓ Убедитесь, что python main.py запущен

  Web App не открывается:
    ✓ Убедитесь, что python run_api.py запущен
    ✓ Проверьте URL в настройках @BotFather
    ✓ Для локального теста используйте ngrok

  Ошибки при установке:
    ✓ Нужен Python 3.9 или выше
    ✓ Создайте виртуальное окружение: python -m venv venv
    ✓ Активируйте: source venv/bin/activate (macOS/Linux)
                    venv\Scripts\activate (Windows)
    ✓ Установите: pip install -r requirements.txt
